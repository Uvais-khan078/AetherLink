# AetherLink - Internet-Free Bluetooth Messaging

![AetherLink Banner](https://via.placeholder.com/800x300/0F172A/22D3EE?text=AetherLink+-+Offline+Messaging)

**AetherLink** is an **offline Bluetooth messaging app** that lets you send messages without internet, Wi-Fi, or mobile data. Perfect for remote areas, disaster situations, college campuses, and privacy-focused communication.

## ✨ Features

- **True Offline Communication** using Bluetooth Classic (RFCOMM)
- **Real-time Chat** between nearby devices
- **Paired Device Connection** (easy device selection)
- **Message History** in chat interface
- **Beautiful Ocean Theme** (Blue-Cyan)
- **Modern UI** with Material Design
- **Simple & Lightweight** – Runs smoothly on low-end devices

## 🎨 Theme
**AetherLink** uses a calm **Blue-Cyan Ocean** color palette:
- Bright Cyan Accents (Relay / Status)
- Blue Sent Messages
- Clean & Modern Look

## 🚀 How to Use

1. Install the app on **two or more Android phones**
2. Turn on **Bluetooth** and **Location** (required for discovery)
3. Open AetherLink on both devices
4. One device will start listening, other can select it from the dropdown
5. Start chatting offline!

> No internet or SIM card required!

## 🛠️ Tech Stack

- **Language**: Kotlin
- **UI**: XML + ViewBinding + Material Components
- **Bluetooth**: Classic Bluetooth (RFCOMM Sockets)
- **RecyclerView**: Groupie Library (for chat messages)
- **Architecture**: MVVM-inspired simple structure
